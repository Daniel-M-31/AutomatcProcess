# AutomaticProcess

Systems that manipulate the OS to ensure it **works on your behalf**, 
automating tasks such as scheduled shutdowns, file organization, and backups.

## 🚀 Available Features
* **Scheduled Shutdown:** Managed via the **baku** version of AutomaticProcess.
* **File Organization:** (Coming soon).
* **Automated Backups:** (Coming soon).

## 🐧 Compatibility
* Currently, the tool is **only available for Linux systems**.

## 🛠️ Installation & Usage
1. Download the installer located in the `linux` folder of this repository.
2. Run the file through your terminal using the following command:
   ```bash
   ./installer
   ```
3. The **baku** utility will be automatically installed in `/usr/local/bin`.
4. To start using it, simply type the following command in your terminal:
   ```bash
   baku
   ```
